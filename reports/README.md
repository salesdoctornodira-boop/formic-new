# CCS — Инфографик ҳисобот (опросник аналитикаси)

Бу папка **алоҳида инфографик PDF-ҳисоботни** ва уни яратадиган скриптларни сақлайди.
Асосий иловага (index.html / api / DB) **ҳеч қандай ўзгартириш киритилмаган** — бу мустақил артефакт.

## Файллар
- `CCS_Analytics_<период>.pdf` — тайёр PDF ҳисобот (**асосий натижа**).
- `CCS_Analytics_<период>.html` — ўша ҳисоботнинг манба HTML (браузерда очиб, қайта чоп этиш/таҳрирлаш мумкин).
- `analyze.py` — `data/ccs.db`дан ўқиб, ҳамма аналитикани ҳисоблайди (`data.json`).
- `generate_report.py` — `data.json`дан инфографик HTML тузади.
- `make_report.sh` — бутун жараённи бир буйруқда бажаради.

## Қайта ишлаб чиқариш
```bash
bash reports/make_report.sh
```
Скрипт `data/ccs.db`нинг **вақтинчалик нусхасидан** ўқийди — live DB'га ёзмайди, охирида checksum'ни текширади.

## Ҳисоблаш усули (методология)
- **DCS (общий балл)** — вазнли композит: Final (раҳбарлар) ва Peer (ҳамкасблар) `weightsConfig`даги
  `finalShare`/`execShare` бўйича аралаштирилади. Барча логика `index.html`даги
  `weightedDCS` / `poolScore` / `getMandatoryTargets` билан айнан бир хил (Python'га кўчирилган).
- **Средний балл** — 4 қадрият бўйича оддий ўртача. **Всего баллов** — олинган ҳамма балл йиғиндиси.
- **Охват (coverage)** — баҳолаган / баҳолаши шарт бўлганлар (`eligibleMap`).
- **Rejected** баҳолар балл сифатида `rejectScore` (=5) ҳисобланади; **skipped** («билмайман») чиқарилади.

## Талабга кўра қабул қилинган қарорлар
1. **Раҳбарлар исм-фамилияси билан** — Final (раҳбар) баҳо/изоҳлари муаллифи исм-фамилия ва лавозим
   билан кўрсатилади (раҳбарлар аноним эмас). **Оддий ходимлар аноним** қолади (агрегация: «матн» × сон).
2. **Изоҳлар — фақат реал/честнийлари** — фақат модерациядан ўтган (`approved`), матни бор изоҳлар.
   `rejected` (модератор рад этган) ва `pending` (текширилмаган) чиқарилади.
3. **ТОП-15** — «ўртача балл × охват» бўйича (энг кўп одамдан энг яхши балл), мин. 5 баҳо билан.
   Топ-раҳбарлар (Executive) фақат раҳбарлар томонидан баҳоланади → алоҳида блокда.
